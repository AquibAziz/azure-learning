## How to add a new course

1. Create a folder: `<platform>-<short-name>/` (e.g., `udemy-scott-duffy-az104`)
2. Add a `README.md` with the course overview (instructor, link, syllabus, progress)
3. One file per session/day: `day-01-topic.md`, `day-02-topic.md`, etc.
4. Use the course log template in [`/CONTRIBUTING.md`](../CONTRIBUTING.md)
5. Update the table above

## Why organize this way

When I learn VNets on YouTube and again on Udemy, the **notes don't duplicate**. Each course folder is a log of what I watched. The distilled understanding lives once, in `01-fundamentals/`. Course notes point there.
