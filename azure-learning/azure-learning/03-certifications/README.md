## Tips that apply to all Azure certs

- Microsoft Learn has **free official learning paths** for every exam — start there
- Take at least 2 full-length practice exams before the real thing
- Hands-on experience beats memorization — use a free Azure account
- The **exam objectives page on Microsoft's site is the source of truth** — everything else is supplementary
