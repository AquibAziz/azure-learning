# AZ-900 — Azure Fundamentals

The entry-level certification. Broad coverage, low depth. Aim to finish in 3–4 weeks of consistent study.

**Official page:** https://learn.microsoft.com/en-us/credentials/certifications/azure-fundamentals/
**Exam duration:** 45 minutes, ~40–60 questions
**Passing score:** 700 / 1000
**Cost:** ~$99 USD (check current pricing and regional discounts)

## Exam objectives (high level)

- [ ] **Describe cloud concepts** (~25%)
  - [ ] Benefits of cloud services
  - [ ] Cloud service types (IaaS, PaaS, SaaS)
  - [ ] Cloud deployment models (public, private, hybrid)

- [ ] **Describe Azure architecture and services** (~35%)
  - [ ] Core architectural components (regions, AZs, resource groups, subscriptions)
  - [ ] Azure compute & networking services
  - [ ] Azure storage services
  - [ ] Azure identity, access, security

- [ ] **Describe Azure management and governance** (~30%)
  - [ ] Cost management
  - [ ] Governance features (tags, policies, locks)
  - [ ] Monitoring tools

## Study plan (4 weeks)

### Week 1 — Cloud concepts
- Read MS Learn: "Describe cloud concepts" module
- Watch AZ-900 intro videos
- Write notes on: IaaS vs PaaS vs SaaS, shared responsibility model, CapEx vs OpEx

### Week 2 — Core Azure services
- VNet, VM, Storage, Databases — at a high level
- Hands-on: create a free account, spin up one VM, delete it
- Write notes: when to use which compute service

### Week 3 — Identity, security, governance
- Microsoft Entra ID, RBAC, MFA
- Azure Policy, locks, tags, cost management
- Azure Trust Center, compliance offerings

### Week 4 — Practice & review
- Take 2+ full practice exams
- Review weak areas
- Re-read cheat sheets
- Book & take the exam
