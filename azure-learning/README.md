## 📖 How this repo is organized

This is a **hybrid knowledge base** — topic-based canonical notes plus a course-by-course learning journal.

| Folder | Purpose |
|---|---|
| [`01-fundamentals/`](./01-fundamentals/) | **Knowledge base** — clean, canonical notes per Azure topic |
| [`02-courses/`](./02-courses/) | **Learning journal** — day-by-day notes from each course I take |
| [`03-certifications/`](./03-certifications/) | Cert-specific study plans, cheat sheets, practice questions |
| [`04-hands-on-labs/`](./04-hands-on-labs/) | Lab write-ups with commands, screenshots, gotchas |
| [`05-projects/`](./05-projects/) | Bigger end-to-end builds (portfolio pieces) |
| [`06-cheat-sheets/`](./06-cheat-sheets/) | Quick reference sheets (CLI, ports, services) |
| [`07-flashcards/`](./07-flashcards/) | Q&A decks for spaced repetition |
| [`99-resources/`](./99-resources/) | Books, blogs, people to follow, useful links |

**The rule:** course notes are raw and chronological. Fundamentals are distilled and canonical. Course notes *link into* fundamentals — they don't duplicate.
