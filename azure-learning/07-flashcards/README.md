## How to use these

1. Read the question
2. Answer it out loud or in your head **before** expanding the answer
3. Expand, check, and honestly rate: got it / close / nope
4. Rotate through decks with spacing — harder cards more often

## Format

```markdown
### Q: Question text goes here

<details><summary>Answer</summary>

Answer goes here.

</details>
```

Markdown's `<details>` tag is supported on GitHub and lets the answer collapse by default — perfect for self-testing without accidentally seeing the answer.

## Tools worth considering

- **Anki** — export these to Anki if you want real spaced repetition with a scheduling algorithm
- **Obsidian** — if you move to Obsidian, there are SRS plugins
